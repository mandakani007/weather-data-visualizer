## About
This project analyses a small real-like weather dataset (Aug–Nov 2024, 120 days). It demonstrates data loading, basic cleaning, simple statistics using NumPy, grouping and aggregation with Pandas, and visualizations with Matplotlib.

---

## Folder Structure
- `data/` - raw and cleaned CSV files
- `images/` - saved PNG plots
- `src/` - Python script (student-style)
- `weather_analysis_practice.ipynb` - Jupyter notebook version
- `report/Summary_Report.md` - brief report with findings
